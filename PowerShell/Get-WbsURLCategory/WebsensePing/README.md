# WebsensePing

Folder **"WebsensePing"** should contain the following files:
* btrow42.dll
* btuc42.dll
* diagnostics.cfg
* Diagnostics.dll
* fpblowfish.dll
* libcurl.dll
* libeay32.dll
* log4cxx.dll
* msvcp110.dll
* msvcr110.dll
* ssleay32.dll
* WebsensePing.exe
* WFCBase.dll
* WFCCrypto.dll
* WFCNetwork.dll
* WFCRoot.dll


By default, the files are located here **"C:\Program Files\Websense\Web Security\bin"**
